If you are new to the Linux environment, here are some instructions for using the text editors vim and nano:

To use nano:
type "nano myfile.py"
look up some shortcuts to make your life easier
quit the program using controlX
run it from the terminal using (python3 myfile.py)

To use vim:
type "vim myfile.py"
to begin writing, press "i" or the insert button
look up some shortcuts to make your life easier
to save and exit, hit escape then colon wq (:wq)
to not save and exit, hit escape then colon q! (:q!)
run it from the terminal using (python3 myfile.py)
